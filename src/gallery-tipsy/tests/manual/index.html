YUI3 Tooltip and Popover

Contextual Tooltips and popovers

What is it?

Y.Tipsy is a tooltip for the YUI3 Library.

Fork from Github

What features are supported?

Currently Y.Tipsy tooltip supports the following:

  • Placement of tooltip above, below, left or right of content
  • Offset support (how far the tooltip should be from the content (broken)
  • Ability to set placement and tooltip content through data-attributes on DOM elemnts
  • Customizable triggers (events that should show or hide the tooltip)
  • Animation (no customization yet)
  • The cool "tip" that points to the content
  • Totally skinnable
YTipsy is built as a Y.Node plugin and it's goal is to be as lightweight as possible.

What about the popover?

Y.Popover extends Y.Tipsy and has the feature set of Y.Tipsy. In addition it currently supports passing in a title. Additional features are planned. Let me know if you have any ideas on Github.
